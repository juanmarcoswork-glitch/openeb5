Cambios respecto a v1 (producción actual) — actualizado 2026-06-14 tras feedback

  • Top bar: añadido <li> de teléfono junto al email, separado por línea vertical sutil. Ícono verde marca #44ba74. Hover a blanco/verde. Inyectado server-side via ob_start en template_redirect (cero CLS, cero JS, compatible con NitroPack).
  • Hero padding superior (root cause encontrada): el theme padre Consultio aplica #ct-header-wrap.ct-header-layout25 { margin-bottom: -130px } (línea 8244 de themes/consultio/assets/css/theme.css) para hacer overlay transparente sobre el primer hero — comportamiento intencional del theme. v1 con padding-top: 0 + min-height: 560px dejaba el contenido del hero completamente bajo el menú. v2 usa padding: 240px 0 120px; min-height: 720px compensando: 40px top bar + 130px main header + 70px holgura visual = 240px. Mobile: padding: 130px 0 80px (el header colapsa).
  • Tipografía: title pasa de clamp(34px, 5.2vw, 62px) a clamp(38px, 5.6vw, 68px). Sub pasa de 15-18px a 16-19px. Eyebrow de 12px a 13px.
  • Eyebrow con bullet verde marca: punto #3dcf86 con halo translúcido antes del texto, anclando el hero a la identidad visual del header (donde el verde es el color del menú activo).
  • Acento "Green Card" → verde marca (cambiado tras feedback). El highlight del title ya no es dorado #e3b770 sino verde marca #3dcf86. El dorado se reserva para el CTA primario "Schedule a Free Consultation" (jerarquía visual: verde = identidad, dorado = acción).
  • Prop values row (nuevo tras feedback): 3 props con icono SVG inline + texto, en círculo translúcido verde marca. Editable vía atributos prop1_text / prop2_text / prop3_text del shortcode. Defaults: USCIS-Approved Regional Center, Real Estate-Based Projects in Texas, End-to-End Investor Support. En mobile pasa a column layout.
  • CTA #2 → llamada directa: sustituye "Read the EB-5 guide" por "1 - 855 - OPENEB5" con icono teléfono SVG inline. Borde verde marca, hover llena con verde y texto navy. href="tel:19567335390".
  • Overlay más rico: gradiente pasa de (7,26,48,.78 → .30) a (7,26,48,.86 → 11,42,73,.32) con stop intermedio — más profundidad sin ocultar la imagen.
